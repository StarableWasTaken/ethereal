What Is Ethereal Launcher?

Ethereal Launcher offers an alternative Minecraft experience by focusing on a streamlined interface, Microsoft OAuth sign-in, and faithful use of official Mojang assets. The launcher never redistributes the game, bypasses authentication, or launches accounts that do not already own Minecraft.

How the Launcher Works

1. Microsoft Sign-In

Users authenticate via the official OAuth window using the openid, profile, offline_access, and XboxLive.signin scopes.

2. Ownership Verification

The launcher forwards the token to the backend, which performs Xbox Live, XSTS, and Minecraft Services checks before returning the username, UUID, and access token.

3. Official Downloads

Version manifests and files are fetched directly from piston-meta.mojang.com and piston-data.mojang.com with zero modifications.

4. Standard Launch

Java arguments, classpath, and authentication options mirror the official launcher while invoking the standard net.minecraft.client.main.Main entrypoint.

5. Optional Cosmetics

Community-made capes or textures can be applied via local resource packs. Cosmetics never touch Mojang assets or gameplay balance.

Security & Compliance

  • OAuth-only login — no passwords handled, no legacy accounts supported.
  • Only usernames, UUIDs, and temporary access tokens are stored locally.
  • Backend enforces HTTPS, rate-limits, sanitizes input, and keeps cosmetic data isolated from credentials.
  • Required scopes (openid, profile, offline_access, XboxLive.signin) allow Xbox Live RPS flow, XSTS token creation, and Minecraft ownership validation.
